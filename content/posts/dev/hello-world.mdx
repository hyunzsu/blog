## 블로그를 시작합니다

이 블로그는 Next.js 15 App Router와 MDX로 만들었습니다.

### 기술 스택

- **Next.js 15** - App Router
- **TypeScript** - 타입 안전성
- **Tailwind CSS v4** - 스타일링
- **MDX** - 마크다운 + JSX

### 코드 예시

```typescript
function greet(name: string): string {
  return `안녕하세요, ${name}님!`;
}

console.log(greet("hyunzsu"));
```

앞으로 개발, 일상, 에세이 등 다양한 글을 작성할 예정입니다.
